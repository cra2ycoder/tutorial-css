Borders

Preview:

Border Default:

HTML:

              
                <div class="border-default"></div>

CSS:

              
                .border-default {
                  width: 100px;
                  height: 100px;
      
                  border-left-style: solid;
                  border-right-style: solid;
                  border-top-style: solid;
                  border-bottom-style: solid;
      
                  border-left-width: 1px;
                  border-right-width: 1px;
                  border-top-width: 1px;
                  border-bottom-width: 1px;
      
                  border-left-color: red;
                  border-right-color: red;
                  border-top-color: red;
                  border-bottom-color: red;
                }

Border Group:

HTML:

                
                  <div class="border-group"></div>

                  .border-group {
                    width: 100px;

                    border: 1px solid green;
                  }

Border Radius:

Default:


HTML:

                
                    <div class="border-radius-group"></div>

CSS:

                
                    .border-radius-group {
                      width: 100px;
                      height: 100px;
                      border: 3px solid purple;
                      border-radius: 10px;
                    }

Types of Radius:

top left
top right
bottom left
bottom right

HTML:

                  
                      <section class="border-radius-list">
                          <div class="border-size border-tlr">top left</div>
                          <div class="border-size border-trr">top right</div>
                          <div class="border-size border-blr">bottom left</div>
                          <div class="border-size border-brr">bottom right</div>
                      </section>

CSS:

                  
                      .border-radius-list {
                        display: flex;
                      }
            
                      .border-size {
                        width: 100px;
                        height: 100px;
                        border: 2px solid;
                        border-radius: unset;
                        display: flex;
                        align-items: center;
                        justify-content: center;
                        margin: 5px;
                      }
            
                      .border-size.border-tlr {
                        border-color: red;
                        border-top-left-radius: 30px;
                      }
            
                      .border-size.border-trr {
                        border-color: green;
                        border-top-right-radius: 30px;
                      }
            
                      .border-size.border-blr {
                        border-color: blue;
                        border-bottom-left-radius: 30px;
                      }
            
                      .border-size.border-brr {
                        border-color: violet;
                        border-bottom-right-radius: 30px;
                      }
