Type selectors aka element selectors

Preview:

What color do you like?

I like blue.

I prefer BLACK!

HTML:

              
                  <p>What color do you like?</p>
                  <div>I like blue.</div>
                  <p>I prefer BLACK!</p>

CSS:

              
                  /* All p elements are red */
                  p {
                    color: red;
                  }

                  /* All div elements are blue */
                  div {
                    color: blue;
                  }
